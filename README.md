# PdfViewPager

Android widget to display PDF documents in your Activities or Fragments.

Usage
-----

Use *RemotePDFViewPager* to load from remote URLs

    RemotePDFViewPager remotePDFViewPager =
      new RemotePDFViewPager(context,
         "",
         listener);
